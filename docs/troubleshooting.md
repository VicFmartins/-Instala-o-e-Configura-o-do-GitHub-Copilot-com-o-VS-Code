# Troubleshooting

## O Copilot nao aparece no VS Code

- Confira se as extensoes `GitHub Copilot` e `GitHub Copilot Chat` estao instaladas.
- Verifique se voce entrou com a conta GitHub correta.
- Atualize o VS Code para uma versao recente.

## O chat nao funciona

- Confirme se o VS Code esta atualizado.
- Saia da conta GitHub no VS Code e entre novamente.
- Recarregue a janela com `Developer: Reload Window`.

## Erro de autenticacao

- Faca logout da conta GitHub no VS Code.
- Reabra ou recarregue o editor.
- Faca login de novo com a conta que tem acesso ao Copilot.

## Sugestoes ruins ou excessivas

- Ajuste as configuracoes do workspace.
- Desative sugestoes em `plaintext` e `scminput` se estiver atrapalhando.
- Use instrucoes de repositorio para orientar estilo, seguranca e nivel de detalhe.
