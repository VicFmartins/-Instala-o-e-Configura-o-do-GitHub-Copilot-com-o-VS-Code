# Checklist de Instalacao

## Antes de comecar

- Ter o Visual Studio Code instalado
- Ter uma conta GitHub autenticada no navegador
- Ter acesso ativo ao GitHub Copilot na conta
- Estar usando uma versao recente do VS Code

## Passo a passo

1. Abrir o VS Code.
2. Ir em `Extensions`.
3. Instalar `GitHub Copilot`.
4. Instalar `GitHub Copilot Chat`.
5. Fazer login com a conta GitHub autorizada.
6. Voltar ao editor e confirmar se as extensoes aparecem ativas.
7. Abrir um projeto e testar uma sugestao inline.
8. Abrir o chat do Copilot e fazer uma pergunta simples sobre o codigo.

## Validacao rapida

- Sugestoes aparecem como texto inline
- O chat abre normalmente no VS Code
- A conta correta esta autenticada
- O Copilot responde dentro do editor
